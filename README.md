## How to Launch & Use

### Local Testing
1. Open `index.html` in a modern web browser
2. All features work locally without a server
3. Test responsiveness using browser dev tools

### Deployment Options
1. **Netlify**: Drag and drop the folder
2. **Vercel**: Connect to a Git repository
3. **GitHub Pages**: Push to a repository and enable Pages
4. **Traditional Hosting**: Upload files via FTP

### Customization Tips
1. **Replace Images**: Update paths in HTML to your professional photos
2. **Update Social Links**: Change URLs in navigation and footer
3. **Add Real Content**: Replace placeholder performance data
4. **Connect Form**: Integrate with backend service (FormSpree, Netlify Forms, etc.)
5. **E-commerce**: Connect merchandise to Shopify, WooCommerce, or Stripe
